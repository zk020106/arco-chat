# MarkdownCard Markdown 卡片组件

## 组件简介
用于渲染和展示 markdown 格式的内容，支持代码块、思考块等扩展。

## 使用示例

### 基础用法
```vue
<MarkdownCard content="# 标题\n正文内容" />
```

### 展示代码块
```vue
<MarkdownCard content="```js\nconsole.log('Hello')\n```" />
```

### 展示思考块
```vue
<MarkdownCard content="> 思考：这是一条提示信息" />
```
